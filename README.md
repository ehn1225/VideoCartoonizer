<h2>Video Cartoonizer</h2>
<p>이 프로그램은 Python과 OpenCV를 이용하여 만든 동영상 만화화 프로그램 입니다..</p>

<h2>Preparation</h2>
이 프로그램을 사용하기 위해서 python과 OpenCV가 필요합니다.
<p>pip install opencv-python opencv-contrib-python</p>
